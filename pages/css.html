O que é:

CSS é a linguagem responsável por definir o visual do HTML — cores, fontes, espaçamento, tamanhos, alinhamento, etc.

Função:

Transformar a estrutura HTML em uma interface bonita, organizada e responsiva.

Formas de usar:

Inline: dentro da própria tag
p style="color: blue;">Texto azul/p>
Interno: dentro do style> no head>
style>
p { color: red; }
/style>
Externo: em arquivo separado (style.css)
link rel="stylesheet" href="style.css">

Conceitos importantes:

Seletores: definem o que será estilizado (p, .classe, #id).
Propriedades: definem o estilo (ex: color, font-size, margin).
Box Model: estrutura de cada elemento (conteúdo, padding, borda, margem).
Responsividade: uso de @media para adaptar a tela.

Importância:

O CSS permite que um site seja atraente, legível e compatível com qualquer dispositivo.
