O que é:

React é uma biblioteca JavaScript criada pelo Facebook (Meta) para construir interfaces de usuário (UI) de forma eficiente e modular.

Função:

Facilitar o desenvolvimento de componentes reutilizáveis e atualizações rápidas sem recarregar a página inteira.

Exemplo simples:

function Ola() {
return h1>Olá, mundo!/h1>;
}
export default Ola;

Conceitos importantes:

Componentes: partes independentes da interface.
JSX: mistura HTML e JavaScript.

Props: parâmetros que passam dados entre componentes.
State: armazena e atualiza dados dentro dos componentes.
Virtual DOM: atualiza apenas o que muda na tela, deixando tudo mais rápido.

Importância:

React é um dos frameworks mais usados no mundo, base para sites como Facebook, Instagram e Netflix.
É essencial para quem quer trabalhar com front-end moderno.
