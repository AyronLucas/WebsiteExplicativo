O que é:

HTML é a linguagem de marcação usada para estruturar o conteúdo de uma página web.
Ele não é uma linguagem de programação, e sim uma estrutura base onde outros recursos (como CSS e JavaScript) atuam.

Função:

Organizar e definir o que aparece na página, como textos, imagens, vídeos, botões, links e formulários.

Estrutura básica:

!DOCTYPE html>
html lang="pt-BR">
head>
meta charset="UTF-8">
meta name="viewport" content="width=device-width, initial-scale=1.0">
title>Título da Página/title>
/head>
body>
h1>Bem-vindo!/h1>
p>Este é o meu primeiro site em HTML./p>
/body>
/html>

Elementos essenciais:

head — informações da página (título, estilos, meta tags).
body — conteúdo visível da página.
h1 a h6 — títulos e subtítulos.
p — parágrafos.
a — links.
img — imagens.
ul / ol / li — listas.
div> / section> — organização e agrupamento de conteúdo.
form> / input> — formulários e campos de entrada.

Importância:

Sem HTML, nenhuma página web existiria. Ele é o esqueleto do site e o primeiro passo para qualquer desenvolvedor.
