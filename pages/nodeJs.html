O que é:

Node.js é um ambiente de execução JavaScript no servidor, criado por Ryan Dahl em 2009.
Ele permite que o JavaScript, antes usado só no navegador, também rode no back-end.
Função:

Construir servidores, APIs, sistemas em tempo real e aplicações completas com uma só linguagem.

Exemplo básico:

const http = require('http');
http.createServer((req, res) => {
res.end('Servidor rodando com Node.js!');
}).listen(3000, () => {
console.log('Servidor ativo em http://localhost:3000');
});

Conceitos importantes:

npm (Node Package Manager): gerencia bibliotecas e dependências.
Módulos: permitem dividir o código em partes.
Express.js: framework para criar servidores e APIs rapidamente.
Assincronismo: executa várias tarefas sem travar o servidor.

Importância:

Com o Node.js, o desenvolvedor pode usar JavaScript em todo o projeto — front e back-end — tornando o trabalho mais integrado e produtivo.
