O que é:

JavaScript é uma linguagem de programação usada para criar interatividade e dinamismo nas páginas web.

Função:

Permitir ações como cliques, animações, validações de formulários, atualizações sem recarregar a página (AJAX), etc.

Exemplo básico:

button onclick="mostrarMensagem()">Clique aqui/button>
script>
function mostrarMensagem() {
alert("Olá! Você clicou no botão!");
}
/script>

Conceitos importantes:

Variáveis: armazenam dados (let, const, var).
Funções: blocos de código reutilizáveis.
Eventos: reagem a ações do usuário (cliques, teclas, etc).
DOM (Document Object Model): permite manipular o HTML via código.
Condicionais e loops: controlam a lógica do programa.

Importância:

É o cérebro da web, responsável por toda interação que o usuário faz em uma página moderna.
